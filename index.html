Deriving Equation of Motion

Parameters:

$T_0$: Tension (Newtons)

$\rho$: Linear String Density (kg / m)

$y(x, t)$: String Displacement

Assumptions:

Infinite Length

Constrained to Transverse Motion (i.e. only move in y-axis)

Under Tension $T_0$ in x_axis

Deriving Equation of Motion

Show String

Show in tiny detail

$F_R = T_0sin\theta_{x + \Delta x} ;\quad sin\theta \approx tan\theta (\text{for small} \theta)$

Deduce that $tan\theta = \frac{\partial y}{\partial x}$

$F_R = T_0\frac{\partialy}{\partialx}\bigg\vert_{x_1 + \Delta x}$ therefore $F_L = -T_0\frac{\partialy}{\partialx}\bigg\vert_{x_1}$

$F = ma ; \quad F = F_L + F_R; \quad a \Rightarrow \frac{\partial^2y}{\partial t^2} ; \quad m_{\Delta x} = \rho \Delta x$

$m\frac{\partial^2y}{\partial t^2} = F_R + F_L$

$m\frac{\partial^2y}{\partial t^2} = T_0\frac{\partialy}{\partialx}\bigg\vert_{x_1 + \Delta x} -T_0\frac{\partialy}{\partialx}\bigg\vert_{x_1}$

$\rho \Delta x\frac{\partial^2y}{\partial t^2} = T_0\frac{\partialy}{\partialx}\bigg\vert_{x_1 + \Delta x} -T_0\frac{\partialy}{\partialx}\bigg\vert_{x_1}$

$\rho \Delta x\frac{\partial^2y}{\partial t^2} = T_0\left(\frac{\partialy}{\partialx}\bigg\vert_{x_1 + \Delta x} - \frac{\partialy}{\partialx}\bigg\vert_{x_1}\right)$

$\rho \frac{\partial^2y}{\partial t^2} = T_0\frac{\left(\frac{\partialy}{\partialx}\bigg\vert_{x_1 + \Delta x} - \frac{\partialy}{\partialx}\bigg\vert_{x_1}\right)}{\Delta x}$

Remember: $\frac{dy}{dx} = \lim\limits_{h \to 0} \frac{y(x + h) - y(x)}{h}$

Remember: $\frac{dy}{dx} = \lim\limits_{\Delta x \to 0} \frac{\frac{\partial y(x_1 + \Delta x)}{\partial x} - \frac{\partial y(x_1)}{\partial x}}{\Delta x}$

$\rho \frac{\partial^2y}{\partial t^2} = T_0\frac{\left(\frac{\partialy}{\partialx}\bigg\vert_{x_1 + \Delta x} - \frac{\partialy}{\partialx}\bigg\vert_{x_1}\right)}{\Delta x}$

$\rho \frac{\partial^2y}{\partial t^2} = T_0\frac{d}{dx}\frac{\partial y(x_1)}{\partial x} \Rightarrow \frac{\partial^2 y(x_1)}{\partial x^2}$

$\rho \frac{\partial^2y}{\partial t^2} = T_0\frac{\partial^2 y(x_1)}{\partial x^2}$

$\frac{\partial^2y}{\partial t^2} = c^2\frac{\partial^2 y(x_1)}{\partial x^2}$ defining $c = \sqrt{\frac{T_0}{\rho}}$

Deriving Modes

Solutions for 1D Wave

$y(x, t) = y(x)e^{i\omega t}$ into $\frac{\partial^2y}{\partial t^2} = c^2\frac{\partial^2 y(x_1)}{\partial x^2}$

$y(x)(i\omega)^2e^{i\omegat} = c^2\frac{\partial^2y}{\partial x^2}e^{i\omega t}$

$y(x)(-\omega)^2e^{i\omegat} = c^2\frac{\partial^2y}{\partial x^2}e^{i\omega t}$ divide $c^2e^{i\omega t}$ or multiply $\frac{1}{c^2e^{i\omega t}}$

$y(x)frac{(-\omega)^2}{c^2}\cancel{e^{i\omegat}} = \cancel{c^2}\frac{\partial^2y}{\partial x^2}\cancel{e^{i\omega t}}$

$y(x)frac{(-\omega)^2}{c^2} = \frac{\partial^2y}{\partial x^2}$

$\frac{\partial^2y}{\partial x^2} = y(x)frac{(-\omega)^2}{c^2}$ : Equation of Motion (Time Harmonic)

Deriving Modes

Solutions for Time Harmonic

$\frac{\partial^2y}{\partial x^2} = y(x)frac{(-\omega)^2}{c^2}$ : Equation of Motion (Time Harmonic)

$y = A e^{ikx} + B e^{ikx}$ : General Solution $k \triangleq \frac{\tau}{\lambda}$ 'Spatial angular frequency' or 'angular wave number' (number of equal phase locations per meter)

$\frac{\partial^2}{\partial x^2}\left(A e^{ikx} + B e^{-ikx}\right) = \frac{-\omega^2}{c^2}\left(A e^{ikx} + B e^{ikx}\right)$

$-k^2A e^{ikx} - k^2 B e^{-ikx} = \frac{-\omega^2}{c^2}\left(A e^{ikx} + B e^{ikx}\right)$

$-k^2\left(A e^{ikx} + B e^{-ikx}\right) = \frac{-\omega^2}{c^2}\left(A e^{ikx} + B e^{ikx}\right)$ divide by $A e^{ikx} + B e^{ikx}$

$-k^2 = \frac{-\omega^2}{c^2}$

$-k^2c^2 = -\omega^2$

$k^2c^2 = \omega^2$

$\omega^2 = k^2c^2$ sqrt both sides

$\omega = kc$: dispersion relation and $k = \frac{\omega}{c}$

$y(x,t) = y(x)e^{i\omega t}$

$y(x,t) = Ae^{ik t}e^{i\omega t} + Be^{-ik t}e^{i\omega t}$

$y(x,t) = Ae^{i(\omega t + kx)} + Be^{i(\omega t - kx)}$
